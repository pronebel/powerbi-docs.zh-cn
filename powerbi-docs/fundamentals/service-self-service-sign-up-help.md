# <a name="what-to-do-if-sign-up-is-disabled"></a>禁用注册后应执行的操作

你已尝试注册 Power BI 并收到了注册禁用的消息。 由于各种原因，某些组织会阻止成员进行自助服务注册。  例如，你的组织可能制定了一项策略，即所有许可证和订阅均由集中式 IT 部门或支持人员管理（甚至是免费许可证）。
